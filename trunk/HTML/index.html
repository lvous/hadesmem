Introduction

HadesMem is a C++-based memory hacking library for Windows based applications, with the goal of providing a safe, generic, powerful, and efficient API. All components (with some minor exceptions, such as the Patcher) are useable on remote targets. Furthermore, all components compile for (and work natively on) both IA32 and AMD64.

To aid in rapid development, HadesMem ships with full Python bindings, useable for anything from the simplest of tests, to full-blown applications.

Features include:

  • Disassembler
  • Pattern scanner
  • Injector
  • Manual mapper
  • Memory modification
  • Remote function caller
  • Module enumeration
  • Code detouring/patching
  • Memory region enumeration
  • Memory scanner
  • Symbol lookup
  • PE file format wrapper
