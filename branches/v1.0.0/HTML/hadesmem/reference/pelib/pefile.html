Description

The PeFile class provides the most basic PE file manipulation functionality, and is the basis on which all other PeLib components in HadesMem are built.

PeFile public types 

class Error : public virtual HadesMemError;

Exception type thrown by PeFile methods when an error occurs. 

enum FileType;

PE file 'type', specifying how the PE file is mapped in memory. Affects RVA to VA conversions.

  • FileType_Image - PE file is mapped as an image (e.g. by LoadLibrary).
  • FileType_Data - PE file is mapped as data (e.g. in it's original on-disk representation).
